# Audit, candidate, and approval artifact contract

The agent owns these artifacts. Merchants review the rendered HTML report and should not be asked to hand-write JSON.

## `audit.json`

Generated by the deterministic helper. It contains `article`, `audit`, `preview`, `status`, and `next`. Treat the Article `body` as the source version.

## `candidate.json`

The agent may start from `audit.json` and add the proposed version:

```json
{
  "status": "candidate-ready",
  "article": {
    "id": "gid://shopify/Article/000000000000",
    "title": "Current article title",
    "handle": "current-handle",
    "storefrontUrl": "https://store.example/blogs/news/current-handle"
  },
  "candidate": {
    "body": "<nav id=\"article-toc\">...</nav><h2 id=\"section\">...</h2>",
    "summary": "Optional summary only when approved as part of this plan.",
    "changes": [
      { "area": "TOC", "summary": "Added stable heading IDs and a clickable table of contents." },
      { "area": "FAQ", "summary": "Added evidence-supported questions and visible answers." }
    ],
    "updates": { "body": "<candidate HTML>" }
  },
  "research": [
    { "url": "https://example.org/source", "title": "Authoritative source", "supports": "Claim supported and limitation." }
  ],
  "eeat": {
    "score": 2,
    "findings": [],
    "blockingIssues": [],
    "confidence": "medium"
  },
  "preview": {
    "mode": "theme-like-fallback",
    "accessState": "password-protected",
    "storefrontUrl": "https://store.example/blogs/news/current-handle",
    "note": "The live storefront could not be verified."
  }
}
```

The candidate must preserve Article identity. `candidate.updates` may contain only `body` and optional `summary`; the helper ignores other write fields.

## `approved-plan.json`

After the merchant explicitly approves the combined report, copy the candidate and add one approval marker:

```json
{
  "approved": true,
  "approval": {
    "confirmed": true,
    "scope": ["body"],
    "note": "Merchant approved the combined audit and preview report."
  },
  "articleId": "gid://shopify/Article/000000000000",
  "updates": {
    "body": "<candidate HTML>"
  }
}
```

Never place access tokens, client secrets, session cookies, private API responses, or merchant credentials in any artifact.
